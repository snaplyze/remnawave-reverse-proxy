## Установка скрипта

:::danger[Важно]
Скрипт нужно запускать с **правами root**, а так же на сервере, на котором будет установлена нода Remnawave.
:::

Введите данную команду на вашем сервере:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Установка ноды

<Steps>

1. Выберите пункт `Установка компонентов Remnawave`

    ![Main menu RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/main_menu.webp)

2. Затем выберите пункт `Установить только ноду`

    ![Install node RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/install_node.webp)

3. Зайдите в панель управления, перейдите в раздел `Ноды` → `Управление`, выберите нужную ноду и найдите поле `Secret Key` (SECRET_KEY). Скопируйте сертификат, используя иконку копирования.

    ![SSL_CERT RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/ssl_cert.webp)

4. Скрипт запросит домен `selfsteal`, который вы указали при первоначальной установке панели или через пункт [`Добавить ноду в панель`](https://wiki.egam.es/ru/installation/add-node/), ip-адрес и сертификат панели.

    :::danger[Важно]
    Не забудьте добавить DNS-запись для вашего домена.
    :::

    ![Menu node end RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/menu_node_end.webp)

5. Затем скрипт запустит ноду и подключит её к панели.

    ![Menu node end2 RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/menu_node_end2.webp)

</Steps>
