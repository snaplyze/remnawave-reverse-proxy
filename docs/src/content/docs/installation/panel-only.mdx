## Running the script

:::danger[Warning]
Script must be run as **root**.
:::

Enter the command on your server:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Install only the panel

<Steps>

1. Select `Install Remnawave components`

    ![Main menu RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/main_menu.webp)

2. Then select `Install only the panel`

    ![Install panel RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/install_panel.webp)

3. After the process is complete, the script will provide all the necessary login information.

    ![Menu panel end RU](https://raw.githubusercontent.com/snaplyze/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/menu_panel_end.webp)

</Steps>

:::danger[Warning]
After the installation is complete, the script will display a unique URL for logging into the control panel.  
**Be sure to save this URL.** It contains the secret key needed for access.

Example login link:
```
https://p.example.com/auth/login?SECRET_KEY=SECRET_KEY
```
Without this URL, **you will not be able** to log into the panel.
:::

## Updating the script

Make sure to update the script before running things in it. You can do this by selecting `Check for updates script` in the script menu or by viewing the GitHub repository for updates.
